## Approval Checklist

Before writing begins:
- [ ] Primary question is specific and real consumers ask it
- [ ] Short answer is clear, concise, and under 150 words
- [ ] Duplication check complete — page doesn't cannibalize existing content
- [ ] All required schema types identified
- [ ] 2-3 unique examples planned (not copy-pasted from other pages)
- [ ] Internal link targets identified
- [ ] Authoritative sources identified
- [ ] Content scoring targets meet minimum thresholds for page type

---

## Post-Writing Validation

After the page is written:
- [ ] Content follows the authority page template structure
- [ ] All planned examples included
- [ ] FAQ section present with FAQPage schema
- [ ] Internal links implemented
- [ ] Sources cited
- [ ] Design system compliance verified
- [ ] Quality gates passed (from `brain/quality-gates.md`)
- [ ] Content score meets or exceeds targets
